# Supabase + Vue + PrimeVue Training App

Link to Supabase's quickstart guide for setting up a Vue project:
https://supabase.com/docs/guides/getting-started/quickstarts/vue


## To get this running in dev mode:

Set up your Supabase project

Create file: /.env.local

Insert the keys into .env.local using the quickstart guide. 
You should be logged in, so hopefully you can see your own project keys in the guide UI

Type in console:
npm run dev
